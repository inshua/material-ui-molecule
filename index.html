What is MaterialRipple.js?

MaterialRipple lets you use that ultra cool active effect on every element you want with modern CSS3 Animations and super smooth animations. You can edit your ripple per CSS, add custom ripples for custom elements and much more!

How to use:

Just implement the jquery.materialripple.js into your project and call the function on any element you want to use it on. For example we called it on .ripple.

$(function(){
	$('.ripple').materialripple();
});

Options

rippleClassAllows you to change the class of the appended ripple effect.
